## Description

### 基本情報

- **ロール**: Physical Attacker（物理アタッカー）
- **タイプ**: All-Rounder（バランス型）
- **攻撃範囲**: Melee（近接）
- **難易度**: Intermediate（中級者向け）

### 初期ステータス（レベル 1）

- **HP**: 3200
- **Attack**: 150
- **Defense**: 80
- **Sp. Attack**: 20
- **Sp. Defense**: 60
- **Critical-Hit Rate**: 0%
- **CD Reduction**: 0%
- **Lifesteal**: 0%（ライフスティールは通常攻撃にのみ適用）

**注記**: 通常攻撃、あんこくきょうだ、すいりゅうれんだの 3 回目のチェーン使用でクリティカル攻撃が可能。ウーラオスのクリティカル率はレベル 5 とレベル 9 で上昇する。

### 進化

- **コダック（Kubfu）**: レベル 5 で進化

### パッシブアビリティ

#### Inner Focus（せいしんりょく）- コダック時

コダックが受ける妨害の持続時間を 15%短縮する。

#### Unseen Fist（みえないこぶし）- ウーラオス時

シールド効果が有効な相手ポケモンに ダメージを与えた時、シールドに与えたダメージの 30%を追加ダメージとしてそのポケモンの HP に与える。

### 通常攻撃

#### Attack（通常攻撃）

**コダック**: 3 回目の攻撃ごとに強化攻撃となり、射程が増加し、相手ポケモンに突進してダメージを与える。

**ウーラオス（一撃の型）**: 3 回目の攻撃ごとに強化攻撃となり、射程が増加し、相手ポケモンに突進してダメージを与え、ヒット時に 3 秒間シールドを得る。

**ウーラオス（連撃の型）**: 3 回目の攻撃ごとに強化攻撃となり、連続する 2 回の攻撃でダメージを与え、ヒット時に技のクールダウンを 1.5 秒短縮する。

**ダメージ詳細**:

- **基本**: 100% Atk + 0 x (Level - 1) + 0
- **強化（コダック）**: 130% Atk + 0 x (Level - 1) + 0
- **強化（一撃の型）**: 150% Atk + 0 x (Level - 1) + 0
- **シールド（一撃の型）**: 70% Atk + 0 x (Level - 1) + 200
- **強化（連撃の型、2 回攻撃）**: 60% Atk + 0 x (Level - 1) + 0

### 技 1 系統

#### Rock Smash（いわくだき）

- **クールダウン**: 6.00 秒
- **タイプ**: Melee（近接）
- **習得レベル**: レベル 1 または 3
- **説明**: 最大 2 秒間力を溜め、溜めている間は受けるダメージを 20%軽減する。1 秒後に最大チャージに達し、放すと指定された方向にパンチを繰り出し、ヒットした相手ポケモンにダメージを与える。この技がキャンセルされるか 2 秒後に放されない場合、0.9 秒のクールダウンが発生する。

**ダメージ**:

- **1 段階チャージ**: 99% Atk + 9 x (Level - 1) + 160
- **2 段階チャージ**: 113.85% Atk + 10 x (Level - 1) + 184
- **3 段階チャージ**: 128.7% Atk + 12 x (Level - 1) + 208
- **4 段階チャージ**: 143.55% Atk + 13 x (Level - 1) + 232
- **5 段階チャージ**: 158.4% Atk + 14 x (Level - 1) + 256

#### Wicked Blow（あんこくきょうだ）

- **クールダウン**: 7.00 秒
- **タイプ**: Melee（近接）
- **習得レベル**: レベル 5
- **説明**: 妨害無効状態になり、受けるダメージを 20%軽減し、移動速度が低下しながら最大 2.5 秒間力を溜める（1 秒後にチャージ完了）。放すと指定された方向に強烈な一撃を繰り出し、ヒット時に強化攻撃を準備する。チャージしていない時はクールダウンが 50%短縮され、小さな直線範囲で相手ポケモンにダメージを与え移動速度を減少させる。チャージ時は大きな円錐範囲でより多くのダメージと失った HP に応じた追加ダメージを与える。中心線範囲の相手ポケモンはクリティカル攻撃を受け押し飛ばされる。

**ダメージ**:

- **チャージなし**: 110% Atk + 7 x (Level - 1) + 175
- **チャージ中心**: 175% Atk + 12 x (Level - 1) + 280 + 敵の失った HP の 10%追加ダメージ（野生ポケモンに対して上限 1200）
- **チャージ円錐**: 175% Atk + 12 x (Level - 1) + 280 + 敵の失った HP の 5%追加ダメージ（野生ポケモンに対して上限 600）

- **レベル 11 強化**: チャージ中のダメージ軽減を 40%に強化。チャージして放した時、中心線範囲の相手ポケモンを押し飛ばしとスタンで行動不能にする。

#### Surging Strikes（すいりゅうれんだ）

- **クールダウン**: 11.00 秒
- **タイプ**: Melee（近接）
- **習得レベル**: レベル 5
- **説明**: 前方にダッシュして流れるような動きで攻撃し、ヒットした相手ポケモンにダメージを与える。前回使用から 4 秒以内に最大 2 回まで追加使用可能。3 回目の使用でヒットした相手ポケモンはクリティカル攻撃を受け投げ飛ばされる。各使用時に強化された強化攻撃を準備し、ヒット時に HP を回復する。

**ダメージ**:

- **1 ヒットあたり**: 90.2% Atk + 4 x (Level - 1) + 125
- **回復（2 回分）**: 最大 HP の 2%

- **レベル 11 強化**: この技使用中に受けるダメージを 30%軽減する。

### 技 2 系統

#### Headbutt（ずつき）

- **クールダウン**: 9.00 秒
- **タイプ**: Dash（ダッシュ）
- **習得レベル**: レベル 1 または 3
- **説明**: 頭突きで前方にダッシュし、相手ポケモンにダメージを与えヒット時に押し飛ばす。

**ダメージ**: 99% Atk + 6 x (Level - 1) + 180

#### Throat Chop（しっぺがえし）

- **クールダウン**: 7.00 秒
- **タイプ**: Dash（ダッシュ）
- **習得レベル**: レベル 7
- **説明**: あんこくきょうだを習得した場合に選択可能。強化攻撃を準備し前方にダッシュして、軌道上の相手ポケモンにダメージを与え 0.7 秒間スタンさせる。

**ダメージ**: 179% Atk + 6 x (Level - 1) + 340

- **レベル 13 強化**: あんこくきょうだがヒットした時、この技のクールダウンを 2 秒短縮する。

#### Liquidation（アクアブレイク）

- **クールダウン**: 10.00 秒
- **タイプ**: Melee（近接）
- **習得レベル**: レベル 7
- **説明**: すいりゅうれんだを習得した場合に選択可能。地面を踏みつけ、近くの相手ポケモンにダメージを与え 3 秒間防御を 30%減少させる。ヒット時に 3 秒間シールドを得て、相手ポケモン 1 体ヒットごとにシールドが強化される（最大 3 回）。

**ダメージ**: 97.5% Atk + 6 x (Level - 1) + 165
**シールド（1 ポケモンヒットあたり、最大 3 回）**: 84% Atk + 0 x (Level - 1) + 240

- **レベル 13 強化**: この技のクールダウンを 1 秒短縮する。

### ユナイト技

#### Ebon Fist（漆黒の終拳）

- **クールダウン**: 112 秒
- **エナジー必要量**: 100000
- **タイプ**: Sure Hit（必中）
- **解放レベル**: レベル 9
- **説明**: あんこくきょうだを習得した場合に選択可能。妨害無効状態になり、あんこくきょうだのクールダウンがリセットされる。指定した相手チームのポケモンにダッシュしてスタンさせ、方向指定するかどうか選択する。方向指定しない場合、軽く押してアッパーカットを放ち、ターゲットにダメージを与えて投げ飛ばし着地時にスタンさせる。方向指定した場合、より強力な攻撃でより多くのダメージを与え指定方向に押し飛ばす。押し飛ばされたターゲットは衝突した相手ポケモンに軽減されたダメージを与え投げ飛ばす。

**ダメージ**:

- **基本**: 330% Atk + 12 x (Level - 1) + 615
- **方向指定時**: 440% Atk + 17 x (Level - 1) + 820
- **衝突ダメージ**: 220% Atk + 9 x (Level - 1) + 410

- **ユナイトバフ（6 秒間持続）**: 移動速度 30%増加、CDR30%、最大 HP の 20%のシールド

#### Flowing Fists（蒼流の舞踏）

- **クールダウン**: 112 秒
- **エナジー必要量**: 100000
- **タイプ**: Melee（近接）
- **解放レベル**: レベル 9
- **説明**: すいりゅうれんだを習得した場合に選択可能。最大 10 秒間すいりゅうこんしんバフを発動し、一時的に妨害無効状態になって近くの相手ポケモンにダメージを与え押し飛ばす。再使用すると妨害無効状態になり受けるダメージを 30%軽減しながら、指定した相手チームのポケモンに最低 5 回の連続攻撃を繰り出し、各攻撃でダメージと押し飛ばしを与える。連続攻撃終了後、最終キックを放ち前方の相手ポケモンにダメージを与え押し飛ばす。すいりゅうこんしんバフが有効な間、追撃使用まで相手チームのポケモンに 2 回ヒットするごとに追撃の連続攻撃回数が 1 回増加（最大 10 ヒットで追加 5 回、最大 10 連続攻撃）。

**ダメージ**:

- **初回ヒット**: 37% Atk + 2 x (Level - 1) + 70
- **追撃中のヒット**: 74% Atk + 4 x (Level - 1) + 140
- **最終ヒットと押し飛ばし**: 74% Atk + 4 x (Level - 1) + 140

- **ユナイトバフ（6 秒間持続）**: 移動速度 30%増加、CDR30%、最大 HP の 20%のシールド
